# AI-Powered Marketing Content Generator

## Overview
This project uses generative AI to create customized marketing content for small businesses. It focuses on improving efficiency and personalization in content creation.

## Objectives
- Generate platform-specific marketing content
- Customize tone and audience targeting
- Evaluate output quality using prompt engineering

## Tools & Technologies
- OpenAI API
- Python
- Prompt engineering techniques

## Methods
- Zero-shot prompting
- One-shot prompting
- Role-based prompts
- Structured prompt design

## Key Findings
- Structured prompts significantly improve output quality
- One-shot prompting improves consistency
- Role-based prompting enhances strategic output

## Limitations
- Output variability
- Requires careful prompt design

## Future Improvements
- Fine-tune model on marketing data
- Build user interface for easier use
